## Getting Started

**Node.js**: v21.6.2

1. **install dependencies**

```bash
npm install
```
2. **Set up environment variables**

```bash
GRAPQL_ENDPOINT=http://localhost:3000/graphql
```

3. **Run Application**

```bash
npm run dev
```

Open [http://localhost:3001](http://localhost:3000) with your browser to see the result.
